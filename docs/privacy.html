Privacy, minus the mystery.

Short version:

AI Slop Blocker does not collect, store, sell, or transmit your personal data or browsing history.

What the extension accesses

While enabled, AI Slop Blocker reads visible page content and metadata solely to identify content explicitly labeled as AI-generated, AI-assisted, or synthetic. It hides matched page elements locally.

What is stored

The extension stores only your on/off preference using Chrome's local extension storage. This preference remains on your device.

What is shared

Nothing. The extension does not send page content, browsing history, personal information, or usage analytics to the developer or any third party.

Optional payment link

The landing page and extension popup contain an optional link to a Stripe-hosted payment page. Visiting or using that page is optional and is governed by Stripe's own privacy practices. No payment data is handled by the extension.

Changes

Material changes to this policy will be published on this page and in the public source repository.
