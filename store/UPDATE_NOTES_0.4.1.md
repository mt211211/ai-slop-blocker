# Version 0.4.1 Update Notes

## Chrome Web Store release notes

Adds clearer help and onboarding around the MVP extension.

- Adds a Learn More link in the extension popup that opens the GitHub Pages
  landing page.
- Expands the landing page with a simple feature walkthrough covering the
  toggle, blank-space blocking, dynamic page scanning, personal filters,
  privacy, and optional Big Mac funding.
- Keeps the v0.4.0 blocking improvements for generic websites, X/Twitter, and
  YouTube Shorts.
