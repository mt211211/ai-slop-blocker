# Chrome Web Store Privacy Answers

Use these answers in the Privacy practices tab. Re-check them against the
dashboard wording before submission.

## Single purpose description

AI Slop Blocker hides page content that is explicitly labeled or identified by
known page patterns as AI-generated, AI-assisted, or synthetic.

## Permission justification

### storage

Stores the user's on/off preference locally so the extension remembers whether
blocking is enabled. It also stores the user's personally created filter rules,
such as phrases and content, creator, or channel URLs the user explicitly
chooses to block. These rules stay on the user's device. No browsing history or
page content is collected or transmitted.

### Site access: all websites

The extension's user-facing purpose is to detect and hide explicitly labeled
AI-generated or synthetic content on any website the user visits. Page content
is examined and modified locally only while blocking is enabled. Nothing is
transmitted.

## Data use declarations

- Personally identifiable information: No
- Health information: No
- Financial and payment information: No
- Authentication information: No
- Personal communications: No
- Location: No
- Web history: No collection or transmission
- User activity: No collection or transmission
- Website content: Read locally only to provide the extension's single purpose.
  User-selected phrases and URLs may be stored locally as personal filters;
  nothing is transmitted
- Data sold to third parties: No
- Data used for unrelated purposes: No
- Data used for creditworthiness or lending: No

## Remote code

No. All executable code is included in the extension package.
