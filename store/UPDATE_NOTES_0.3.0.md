# Version 0.3.0 Update Notes

## Chrome Web Store release notes

Adds private personal filters and an interactive “Block as AI” picker.

- Users can select an unlabeled content item and block either that item or its
  detected creator/channel.
- Adds a personal filter manager for phrases, creators, channels, and specific
  content URLs.
- Personal filters automatically apply while browsing and on infinite-scroll
  content.
- All personal filters remain locally on the user's device.
- No remote classifier, account, analytics, or paid API is used.
